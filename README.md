# Cronometro Ciclismo

Aplicacion web para registrar tiempos de competencias de ciclismo.

## Funcionalidades

- Registrar corredores (dorsal, nombre, equipo)
- Cronometro de carrera con precision de milisegundos
- Registro rapido de llegadas con botones por dorsal
- Clasificacion automatica por orden de llegada
- Exportar resultados a CSV (compatible con Excel)
- Guardado automatico en el navegador
- Funciona sin internet
- Responsive (funciona en celular y computadora)

## Como usar

1. Abrir `index.html` en cualquier navegador
2. Configurar la carrera (nombre, categoria, distancia)
3. Registrar los corredores
4. Presionar "INICIAR CARRERA" al dar la salida
5. Registrar las llegadas tocando el boton del dorsal correspondiente
6. Exportar resultados al finalizar

## Archivos

- `index.html` - Pagina principal
- `styles.css` - Estilos visuales
- `app.js` - Logica de la aplicacion
